# AWS CLI Public Key

Replace the value in [aws-cli.gpg](aws-cli.gpg) with the key published under
"Optional: Verifying the integrity of your downloaded zip file" in the Linux section of the
[AWS CLI install and update instructions](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html).

# Google Chrome Public Key

Replace the value in [google-linux-signing-key.pub](google-linux-signing-key.pub) with the key published
under "Linux Package Signing Keys" at [Google Linux Software Repositories](https://www.google.com/linuxrepositories/).
